# FlashCards

## Overview

An intelligent flashcard study application powered by AI to help students learn more effectively.

## Features

- AI-generated flashcards from study materials
- Custom quiz for reinforced learning
- Multi-subject support

## Usage

```bash
npm start
```

## Technologies Used

- React/Node.js
- AI/ML integration
  - Gemini
